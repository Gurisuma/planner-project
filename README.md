# Ежедневник

Сайт предназначен для создания и хранения своих задач в удобном электронном формате. При создании использовались:

1. Флексбокс-вёрстка;
2. Адаптивная вёрстка под разные устройства;
3. Трансформации;
4. Форма;
5. template-карточки;
6. JavaScript.
